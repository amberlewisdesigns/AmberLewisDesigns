alaCart is an iOS app designed to help users find nearby food carts, view a menu, select food items for purchase, and checkout all in one place. Features include a searchable map, sorting and filtering results, and customer ratings and reviews.

inVision Prototype

Challenges and Strategy

alaCart is a feature rich application. The ability to view and order from food menus is an advantage over competing apps. It also means hungry, time-cruched users are at high risk of becoming overwhelmed with options. Creating a clean and organized UI was vital to the app's success.

My Design Process

User Flows

I started by exploring how users would navigate through the app. I created user flows for both new and returning users. The highlighted flow is the most common path a user would take to complete a purchasing cycle. Creating a simple path for the user to complete these steps was central to my mission.
